Here're some of the project's best features:

*   Anime News
*   Anime Info
*   Anime Watch Lists
*   Profiles

<h2>🛠️ Installation Steps:</h2>

<p>1. Clone the repo</p>

```
git clone https://github.com/LabsStar/nanoha-bot
```

<p>2. Install Packages</p>

```
npm i
```

<p>3. Deploy Commands</p>

```
./deploy.bat
```

<p>4. Run the bot</p>

```
npm run start
```
